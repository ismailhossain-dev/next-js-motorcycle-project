# 🏍️ BikeVibe | Premium Bike E-Commerce Store

**BikeVibe** is a high-performance, modern e-commerce platform built using **Next.js (App Router)**. It allows users to browse the latest bikes, view detailed specifications, and make inquiries seamlessly. This project focuses on speed, SEO optimization, and a premium user experience.

## 🌐 Live Demo
[Check out the live site here](https://next-js-bike-project-bd.vercel.app)

---

## 🚀 Key Features
- **Next.js App Router:** Utilizing the latest file-based routing and layout system for optimized performance.
- **Dynamic Bike Showcase:** Dedicated dynamic pages for every bike model with detailed specs.
- **Server-Side Rendering (SSR):** Ensuring lightning-fast page loads and superior SEO rankings.
- **Premium UI/UX:** A sleek dark-themed interface with high-quality glassmorphism effects.
- **Smooth Animations:** Powered by Framer Motion for an interactive and high-end feel.
- **Fully Responsive:** Optimized for a seamless experience across Mobile, Tablet, and Desktop.
- **Contact Integration:** Built-in inquiry system using EmailJS for direct communication.

---

## 🛠️ Technology Stack
- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Database:** MongoDB (via Mongoose)
- **Forms & Alerts:** SweetAlert2
